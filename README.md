每个宏文件是一个 JSON 数组，元素为指令列表或控制结构。常见指令：

- `["kd", "<key>", <ms>]`：按下键（key down），第三项为该动作之后等待的毫秒（整数或 0）。
- `["ku", "<key>", <ms>]`：释放键（key up）。
- `["md", "left|right", <ms>]`：鼠标按下（mouse down），支持 `left`/`right`。
- `["mu", "left|right", <ms>]`：鼠标抬起（mouse up）。
- `["wait", <ms>]`：仅等待指定毫秒（不会注入输入）。
- `["view", [dx, dy], <dur_ms>]`：视角/鼠标相对移动，按 `<dur_ms>` 拆分为若干步逐步移动（用于平滑鼠标移动）。
- `["loop", <count>, [<block>]]`：重复执行块内指令 `<count>` 次（支持嵌套和 `import`）。
- `["import", "subfile.json"]`：在当前位置展开并导入另一个宏文件（支持相对路径）。

示例：`[ ["md","left",0], ["mu","left",120], ["wait",200], ["loop",3, [["kd","q",0],["ku","q",50]]] ]`

使用第三方工具存在封号风险，本仓库内容仅供学习和个人交流使用，风险自负，出事概不负责。
